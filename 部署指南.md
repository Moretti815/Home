# Amis Homepage 部署指南

> 本指南将帮助你快速部署这个基于 Next.js + TypeScript + Tailwind CSS 构建的个人主页项目。

## 📋 项目简介

这是一个功能丰富的个人主页项目，包含以下特性：

- 🎨 现代化设计 - 全屏背景图 + 渐变遮罩 + 动态背景
- ✍️ SVG 绘画标题 + 打字机效果
- 👤 头像动画 + 社交链接
- 📋 关于我卡片 + 精选项目展示
- 📊 技能可视化 + 多语言支持
- 🌓 主题切换 + 时间组件
- 🎵 全局音乐播放器
- 💬 留言板（Waline 集成）
- 🔗 友链页面

## 🚀 部署方式

### 方式一：Docker 部署（推荐）

适合有服务器的用户，一键部署，简单快捷。

#### 1. 准备工作

确保服务器已安装：
- Docker
- Docker Compose

#### 2. 克隆项目

```bash
git clone https://github.com/Moretti815/Home.git
cd Home
```

#### 3. 修改配置

编辑 `config.json` 文件，修改以下关键配置：

```json
{
  "site": {
    "name": "你的主页名称",
    "title": "你的主页标题",
    "url": "https://your-domain.com",
    "author": "你的名字",
    "description": {
      "zh": "中文描述",
      "en": "English description"
    }
  },
  "profile": {
    "name": "你的名字",
    "avatar": "你的头像URL",
    "location": {
      "zh": "你的位置",
      "en": "Your Location"
    }
  }
}
```

#### 4. 构建并启动

```bash
docker-compose up -d
```

#### 5. 访问

打开浏览器访问 `http://localhost:9998`

---

### 方式二：服务器部署（PM2）

适合有 Linux 服务器且需要长期稳定运行的场景。

#### 1. 安装 Node.js 20+

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

#### 2. 安装 PM2

```bash
sudo npm install -g pm2
```

#### 3. 克隆项目

```bash
git clone https://github.com/Moretti815/Home.git
cd Home
```

#### 4. 安装依赖并构建

```bash
npm install
npm run build
```

#### 5. 启动服务

```bash
pm2 start npm --name "homepage" -- start
```

#### 6. 设置开机自启

```bash
pm2 startup
pm2 save
```

### 方式三：Vercel 部署

适合小白用户，免费托管，无需服务器。

#### 1. Fork 项目

访问 [GitHub 仓库](https://github.com/Moretti815/Home)，点击右上角 `Fork` 按钮。

#### 2. 登录 Vercel

访问 [vercel.com](https://vercel.com/)，使用 GitHub 登录。

#### 3. 导入项目

- 点击 `New Project`
- 选择你 Fork 的仓库
- 点击 `Deploy`

#### 4. 绑定域名（可选）

在项目设置中添加自定义域名。

## 🎨 自定义主题

### 打字机效果配置

```json
{
  "typeWriterEffects": {
    "glitchEffect": true,        // 故障效果开关
    "colorGradient": false,      // 颜色渐变开关
    "glitchProbability": 15,     // 故障触发概率 (0-100%)
    "glitchInterval": 1500,      // 故障触发间隔 (毫秒)
    "speedPreset": "medium",     // 速度预设: slow/medium/fast
    "typeSpeed": 100,            // 打字速度
    "deleteSpeed": 50,           // 删除速度
    "pauseTime": 2000            // 暂停时间
  }
}
```

### 组件显示控制

```json
{
  "showSkills": false,           // 显示技能
  "showProjects": true,          // 显示项目
  "showLocalTime": true,         // 显示本地时间
  "showGreetings": true,         // 显示问候语
  "showCustomCursor": true,      // 使用自定义鼠标指针
  "customCursorPath": "/cursors/default.cur"
}
```

## 🖼️ 资源文件

### 图片资源

将以下图片放入 `public/images/` 目录：

| 文件名 | 用途 | 建议尺寸 |
|--------|------|----------|
| `avatar.jpg` | 头像 | 512x512 |
| `index.jpg` | 暗色背景图 | 1920x1080 |
| `index4.jpg` | 亮色背景图 | 1920x1080 |
| `icon.png` | 网站图标 | 512x512 |
| `index2.jpg` ~ `index9.jpg` | 项目展示图 | 1200x630 |

### 音乐文件

将音乐文件放入 `public/music/` 目录，支持 `.mp3`、`.wav`、`.ogg` 格式。

### 鼠标指针

将 `.cur` 文件放入 `public/cursors/` 目录，在配置中指定路径。

## ❓ 常见问题

### Q: 修改配置后没有生效？

A: 重启服务：

```bash
# Docker 部署
docker-compose restart

# PM2 部署
pm2 restart homepage

# 开发环境
# 停止当前服务（Ctrl+C），重新运行 npm run dev
```

### Q: 图片不显示？

A: 检查图片路径：
- 图片放在 `public/images/` 目录
- 配置中使用 `/images/xxx.jpg`（注意开头的 `/`）
- 或使用完整的 URL

### Q: 如何修改端口？

A: 编辑 `.env` 文件：

```env
PORT=8080
```

或修改 `docker-compose.yml`：

```yaml
ports:
  - "8080:9998"
```

### Q: 如何添加新的社交链接？

A: 在 `config.json` 的 `links` 中添加：

```json
{
  "links": {
    "twitter": {
      "url": "https://twitter.com/yourusername",
      "title": { "zh": "Twitter", "en": "Twitter" },
      "icon": "fab fa-twitter"
    }
  }
}
```
